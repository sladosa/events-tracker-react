# Templates — Help

## Što su templati
- Predefinirana struktura Area + Categories + Attributes
- Read-only — ne možeš ih izravno mijenjati
- Dostupni: Health, Fitness, Finance, Work, Personal

## Korištenje templata
1. Structure tab → Edit Mode
2. Klikni "Add Area"
3. Odaberi "From template" radio gumb
4. Odaberi template iz dropdowna
5. Pregledaj preview (broj kategorija i atributa)
6. Klikni "Create from template"

## Što se kopira
- Sve kategorije i podkategorije
- Sve attribute definicije (uključujući suggest opcije)
- Kopirana area je potpuno tvoja — možeš je slobodno mijenjati

## Vidljivost templata
- Structure tab → "Templates" segment: prikazuje dostupne template areae
- Structure tab → "All" segment: vlastite areae + template areae (bez duplicata)
- Template areae s istim slugom kao tvoje vlastite areae se ne prikazuju

## Napomena
- Health template nije vidljiv ako već imaš vlastitu Area s imenom/slugom "health"
- Kopirani template postaje nezavisan od originala (izmjene ne utječu jedne na druge)
